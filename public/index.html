API HOMEPAGE



Products

get all products -> GET https://simplestore.onrender.com/api/v1/product
get a product -> GET https://simplestore.onrender.com/api/v1/product/:id
create a product -> POST https://simplestore.onrender.com/api/v1/product
update a product -> PUT https://simplestore.onrender.com/api/v1/product/:id
delete a product -> DELETE https://simplestore.onrender.com/api/v1/product/:id


Orders

get a order -> GET https://simplestore.onrender.com/api/v1/order/:id
create a order -> POST https://simplestore.onrender.com/api/v1/order
